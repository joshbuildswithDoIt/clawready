The Mac Mini OpenClaw Setup Guide: Local Models + Cloud Fallback

The Mac Mini is the community standard for running OpenClaw 24/7. This guide gets you from unboxing to running — with local models that cut your API bill by 80%.

15 min read · March 2026 · By the ClawReady team

It's quiet, sips power, runs macOS (which means iMessage), and Apple Silicon handles local model inference surprisingly well. By the end of this guide, you'll have a 24/7 agent that costs $5–15/month in API fees instead of $50–150.

What You Need

Hardware

  • Mac Mini with Apple Silicon (M1, M2, M3, or M4)
  • 16GB RAM minimum (24GB+ recommended for larger local models)
  • Ethernet cable (don't rely on WiFi for 24/7 operation)
  • UPS recommended (~$50 battery backup prevents dirty shutdowns during power blips)

1 Prepare Your Mac Mini

Initial Setup

  1. Connect via ethernet — plug directly into your router. WiFi drops kill your agent.
  2. Enable auto-login — System Settings → Users & Groups → Auto Login → your account
  3. Disable sleep — System Settings → Energy Saver → Prevent automatic sleeping
  4. Enable SSH — System Settings → General → Sharing → Remote Login → ON
  5. Set a static IP (or reserve one in your router's DHCP settings)

Enable Headless Operation

Your Mac Mini will run without a monitor, keyboard, or mouse. Prevent display-related issues:

# Prevent sleeping when display is off
sudo pmset -a displaysleep 0 sleep 0 disksleep 0

# Auto-restart after power failure
sudo pmset -a autorestart 1

Install Prerequisites

# Install Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Node.js
brew install node

# Verify
node --version # Should be v20+ or v22+

2 Install OpenClaw

# Install OpenClaw globally
npm install -g openclaw

# Run the setup wizard
openclaw setup

The wizard will ask for:

  • AI provider — pick Anthropic (Claude) for best agent performance
  • API key — get one from console.anthropic.com
  • Default model — start with Sonnet (best price/performance)
  • Messaging channel — WhatsApp, Telegram, Discord, or iMessage

Configure as Service (Auto-Start on Boot)

# Start OpenClaw gateway as a service
openclaw gateway start

# Verify
openclaw gateway status

OpenClaw will now start automatically when your Mac boots and restart if it crashes.

3 Install Ollama (Local Models)

This is where the Mac Mini really shines. Local models = $0 API costs.

# Install Ollama
brew install ollama

# Start Ollama service
brew services start ollama

# Pull a model (Qwen 2.5 7B — excellent for agent tasks)
ollama pull qwen2.5:7b

# Test it
ollama run qwen2.5:7b "Hello, are you working?"

4 Configure Hybrid Model Routing

The magic setup: local model for easy tasks, cloud API for hard tasks. This typically cuts monthly API costs by 70–80%.

  • Default model: Ollama (local) — handles greetings, quick lookups, simple tasks
  • Complex tasks: Claude Sonnet (cloud) — reasoning, coding, long-form writing
  • Heavy lifting: Claude Opus (cloud) — only when you explicitly need the best

5 Set Up Messaging Channels

📱 iMessage (Mac-Only Feature)

The killer feature that makes Mac Mini the preferred hardware. iMessage integration only works on macOS. Text your agent like you'd text a friend — it reads and sends iMessages directly through the Messages app.

💬 WhatsApp

Scan a QR code to link your WhatsApp. Your agent can then receive and respond to WhatsApp messages.

✈️ Telegram

Create a bot via @BotFather, get a token, plug it into OpenClaw config. Most reliable channel — doesn't require QR re-linking.

🎮 Discord

Create a bot in the Discord Developer Portal, add it to your server, configure the token.

7 Security Hardening

Don't skip this. A misconfigured agent is an open door.

  1. API key security — store in .env file, never in committed code
  2. Firewall — enable macOS firewall (System Settings → Network → Firewall)
  3. SSH keys — disable password auth, use SSH keys only
  4. Router — don't expose ports to the internet unless using a tunnel (Cloudflare, Tailscale)
  5. Updates — keep macOS, Node.js, and OpenClaw current

Troubleshooting

Agent not responding after reboot
Check that OpenClaw gateway is running: openclaw gateway status. If not: openclaw gateway restart.
WhatsApp disconnected
WhatsApp links expire periodically. Re-scan the QR code via openclaw setup or the web UI.
Ollama model slow
Check Activity Monitor for memory pressure. If yellow/red, you need a smaller model or more RAM.
High API costs despite routing
Check your model config — make sure the local model is set as default, not just available.
Mac Mini runs hot
Normal under load. Ensure good ventilation. If sustained >95°C, check for runaway processes.
